📊 ИИ-Платформа бизнес-идей
Проект представляет собой Telegram-бота, выполняющего функции персонального бизнес-консультанта на базе искусственного интеллекта. Бот ориентирован на молодёжную аудиторию, помогая проводить анлиз задумок, рассчитывать базовые риски и намечать шаги для создания MVP.

🚀 Основной функционал
Интерактивный интерфейс: Удобное меню с кнопками-шаблонами для быстрого тестирования системы.
Кастомный ИИ-анализ:Интеграция с нейросетью Gemini для генерации структурированных бизнес-отчетов (Анализ монетизации, Риски, Целевая аудитория, План MVP).
Смена позиционирования (Геймификация): Использование динамического системного промпта для адаптации сложных экономических понятий под молодёжный сленг без потери глубины аналитики.

🛠 Технологический стек
Язык разработки: Python 3.10+
Фреймворк:aiogram 3.x(асинхронная библиотека для работы с Telegram Bot API)
ИИ-Интеграция:google-genai (модель gemini-2.5)
Веб-сервер:aiohttp(используется для поддержки механизма Health Check при деплое)
Хостинг/Деплой:Render Cloud (платформа авто-развертывания)

📦 Архитектура и безопасность
Проект спроектирован с учётом лучших практик коммерческой разработки:
1. Безопасность данных (Environment Variables):** Чувствительные данные (`BOT_TOKEN` и `GEMINI_KEY`) полностью вынесены из исходного кода в переменные окружения.
2. Отказоустойчивость:** Реализована конструкция `try-except` для перехвата ошибок API (например, лимитов квот) со стороны Google или Telegram, предотвращающая падение бота.
3.Интегрирован фоновый веб-сервер на aiohttp, который слушает порт хостинга Render, имитируя веб-приложение, что предотвращает принудительную остановку контейнера платформой.
